# 思想内阁系统 (Thought Cabinet System)

## 1. 项目概述

**项目名称**: 思想内阁系统
**项目类型**: 社交/决策类Web应用
**核心功能**: 一个模拟思想辩论和决策的系统，用户可以创建不同的"思想"（观点），让它们相互辩论，最终做出决策。灵感来源于游戏《极乐迪斯科》(Disco Elysium)中的思想内阁系统。
**目标用户**: 喜欢深度思考、角色扮演、以及对复古美学感兴趣的用户

## 2. 视觉与设计规格

### 整体美学方向
- **风格**: 深色复古风格，灵感来自《极乐迪斯科》的颓废、存在主义美学
- **色调**: 以深褐色、暗金色、锈红色为主，营造出老旧、温暖但略带忧郁的氛围
- **质感**: 模拟旧纸张、打印机文字、老式打字机的视觉效果

### 色彩系统
```
背景色:
- 主背景: #1a1814 (深褐色黑)
- 次级背景: #252219 (暗棕)
- 卡片背景: #2d2a23 (暖棕)

主色调:
- 主文字: #d4c5a3 (旧纸色)
- 次级文字: #8b8070 (暗灰棕)
- 强调色: #c9a227 (暗金)
- 警告/激情色: #a34a28 (锈红)
- 技能色: #5a8f7b (暗青绿)
```

### 字体
- **标题**: "Special Elite" (Google Fonts) - 打字机风格
- **正文**: "IBM Plex Mono" (Google Fonts) - 等宽复古字体
- **中文字体**: "Noto Serif SC" - 配合复古感的衬线体

### 响应式断点
- 移动端: < 768px
- 平板: 768px - 1024px
- 桌面: > 1024px

## 3. 页面结构

### 路由结构
```
/ (首页) - 展示思想内阁概述和用户状态
/decision (决策页面) - 创建和管理决策，开展思想辩论
/history (历史记录) - 查看过去的决策历史
/login (登录) - 用户登录入口
/register (注册) - 用户注册入口
```

### 页面布局
- **导航栏**: 固定在顶部，包含Logo、导航链接、登录/注册按钮
- **主内容区**: 各页面具体内容
- **页脚**: 包含版权信息和装饰性文字

## 4. 功能与交互规格

### 首页功能
- 展示"思想内阁"的标题动画效果（打字机效果）
- 展示当前用户的思想内阁状态（如果有用户登录）
- 显示最近的思想/观点卡片
- 引导用户登录或注册

### 决策页面功能
- 创建新的决策/问题
- 添加多个"思想"（观点）到内阁
- 每个思想有:
  - 名称（思想名称）
  - 类型（理性/感性/激进/保守等，用不同颜色标识）
  - 描述（思想的详细说明）
- 思想辩论模拟（随机生成辩论结果）
- 显示每个思想的"支持率"变化

### 历史记录页面功能
- 列出所有过去的决策
- 每个记录显示:
  - 决策标题
  - 做出时间
  - 最终选择的思想
  - 辩论摘要
- 支持按日期筛选

### 登录/注册页面功能
- 简洁的表单设计
- 用户名和密码输入
- 登录后跳转到首页

### 交互效果
- 文字逐字显示的打字机效果
- 卡片悬停时的微妙发光效果
- 页面切换时的淡入淡出
- 按钮点击时的反馈动画

## 5. 组件清单

### Navigation 导航栏
- Logo + 标题
- 导航链接 (首页、决策、历史)
- 登录/注册按钮（或用户状态）

### ThoughtCard 思想卡片
- 显示思想名称、类型、描述
- 支持率进度条
- 悬停发光效果

### DecisionMaker 决策组件
- 问题输入框
- 添加思想按钮
- 开始辩论按钮
- 结果显示区

### HistoryItem 历史记录项
- 日期显示
- 决策标题
- 结果摘要

### AuthForm 认证表单
- 输入框组件
- 提交按钮
- 错误提示

## 6. 技术栈

- **框架**: Next.js 14 (App Router)
- **样式**: Tailwind CSS
- **字体**: Google Fonts (Special Elite, IBM Plex Mono, Noto Serif SC)
- **动画**: CSS Animations + Tailwind Animations
- **状态管理**: React useState/useContext
- **数据存储**: localStorage (客户端)

## 7. 设计细节

### 装饰元素
- 页面边缘的旧纸张纹理效果
- 随机出现的打字机文字装饰
- 复古风格的边框和分隔线

### 动效规格
- 标题打字机效果: 50ms/字符
- 卡片悬停: transform scale(1.02), box-shadow glow
- 页面淡入: opacity 0->1, 300ms ease-out
- 按钮点击: scale(0.95), 100ms

### 响应式适配
- 移动端: 单列布局，简化导航
- 平板: 双列卡片布局
- 桌面: 多列布局，增强视觉效果
